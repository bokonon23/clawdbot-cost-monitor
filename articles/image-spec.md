## Quick Creation Options:

### 1. Canva (Easiest)
- Go to canva.com
- Custom size: 2000x800px
- Search "quote template" or "tech article"
- Paste text, adjust colors
- Download as PNG

### 2. Figma (More Control)
- Create 2000x800 frame
- Add background gradient
- Add text layers
- Export as PNG

### 3. Plain Text PNG (Quick & Ugly)
- Use any image editor
- 2000x800 canvas
- Dark background
- Add quote in large white text
- Export
